OpenCOR uses various APIs, as well as software- and image-based third-party libraries, all of which are listed below, together with the type of license(s) under which they are released, if any.

APIs
  • ﻿ libBioSignalML: a library for reading and writing BioSignalML documents.
    • Location: [OpenCOR]/src/plugins/api/BioSignalMLAPI.
    • Licensing: none.
  • ﻿ CellML API (commit d34b2c8): an interface to manipulate and process CellML documents.
    • Location: [OpenCOR]/src/plugins/api/CellMLAPI.
    • Licensing: MPL v1.1, GPL v2.0 (or later) and LGPL v2.1 (or later).
  • ﻿ libSBML 5.13.0: a programming library to help you read, write, manipulate, translate and validate SBML files and data streams.
    • Location: [OpenCOR]/src/plugins/api/SBMLAPI.
    • Licensing: LGPL v2.1 (or later).
  • ﻿ libSEDML 0.3.1: a library for reading and writing SED-ML models.
    • Location: [OpenCOR]/src/plugins/api/SEDMLAPI.
    • Licensing: BSD 2-Clause.
Software
  • Google Code Prettify: a JavaScript module and CSS file for syntax highlighting of source code snippets in an HTML page.
    • Location: [OpenCOR]/doc/3rdparty/googleCodePrettify.
    • Licensing: Apache v2.0.
  • jQuery 3.1.1: a fast and concise JavaScript library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.
    • Location: [OpenCOR]/doc/3rdparty/jQuery.
    • Licensing: MIT.
  • KaTeX 0.6.0: fast math typesetting for the web.
    • Location: [OpenCOR]/doc/3rdparty/KaTeX.
    • Licensing: MIT.
  • libgit2 0.24.1: a portable, pure C implementation of the Git core methods provided as a re-entrant linkable library with a solid API.
    • Location: [OpenCOR]/src/plugins/thirdParty/libgit2.
    • Licensing: GPL v2.0 with linking exception.
  • LLVM 3.8.1 (incl. Clang): a collection of modular and reusable compiler and toolchain technologies.
    • Location: [OpenCOR]/src/plugins/thirdParty/LLVM.
    • Licensing: UoI/NCSA.
  • OpenSSL 1.0.2h (Windows only): a toolkit implementing the Secure Sockets Layer (SSL v2/v3) and Transport Layer Security (TLS) protocols, as well as a full-strength general purpose cryptography library.
    • Licensing: OpenSSL and SSLeay, which are both BSD-style licenses (more information here).

      we use the binaries that can be found at https://indy.fulgan.com/SSL/.

  • QScintilla 2.9.3: a Qt port of the Scintilla editing component.
    • Location: [OpenCOR]/src/plugins/thirdParty/QScintilla.
    • Licensing: GPL v3.0, but with some exceptions that make it possible to use it under the OpenCOR license.
  • QtSingleApplication: a class that ensures that only one instance of the GUI version of OpenCOR can be run at any given time.
    • Location: [OpenCOR]/src/3rdparty/QtSingleApplication.
    • Licensing: LGPL v2.1 (or later).
  • QtWebKit: a Qt port of WebKit, a web browser engine.
    • Location: [OpenCOR]/src/3rdparty/QtWebKit.
    • Licensing: GPL v2.0 (or later) and LGPL v2.1 (or later).
  • Qwt 6.1.3: a set of widgets and utility classes primarily aimed at programs with a technical background.
    • Location: [OpenCOR]/src/plugins/thirdParty/Qwt.
    • Licensing: Qwt License v1.0 (a slightly less restricive version of LGPL v2.1) and LGPL v2.1 for the MathML text engine, which is based on the QtMmlWidget code.

      we have contributed some code to the MathML renderer, but it has yet to be fully incorporated in the Qwt project. So, in the meantime, we use our version of the MathML renderer, which code can be found here.

  • SUNDIALS 2.7.0: a SUite of Non-linear and DIfferential/ALgebraic equation Solvers (CVODE, IDA and KINSOL).
    • Location: [OpenCOR]/src/plugins/thirdParty/SUNDIALS.
    • Licensing: BSD 3-Clause.
  • web-xslt: XSLT and JavaScript code intended mostly for manipulating MathML and OpenMath.
    • Location: [OpenCOR]/src/plugins/miscellaneous/Core/res/web-xslt.
    • Licensing: W3C, Apache v2.0, MIT, MPL v1.1 or MPL v2.0.

      we currently use only (a slightly improved version of) ctopff.xsl to convert Content MathML to Presentation MathML.

  • zlib 1.2.8: a massively spiffy yet delicately unobtrusive compression library.
    • Location: [OpenCOR]/src/plugins/thirdParty/zlib.
    • Licensing: Zlib.
Images
  • Oxygen 5.26.0: icons used in various parts of OpenCOR.
    • Location: [OpenCOR]/res/oxygen.
    • Licensing: LGPL v3.0.
  • Printable world flags: British and French flag icons used in the Language menu.
    • Location: [OpenCOR]/res/flags.
    • Licensing: none.
  • Qt Creator: icons used in the EditorWidget plugin.
    • Location: [OpenCOR]/src/plugins/widget/EditorWidget/res/qtCreator.
    • Licensing: GPL v3.0, but with an exception that makes it possible to use it under the OpenCOR license.
